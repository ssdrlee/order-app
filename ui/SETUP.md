# 개발 환경 설정 가이드

## Node.js 설치 필요

이 프로젝트를 실행하려면 Node.js가 필요합니다.

### 1. Node.js 설치

1. [Node.js 공식 웹사이트](https://nodejs.org/)에서 LTS 버전을 다운로드하세요.
   - 권장 버전: Node.js 18.x 이상
   - Windows 설치 프로그램(.msi)을 다운로드하여 실행하세요.

2. 설치가 완료되면 터미널(명령 프롬프트 또는 PowerShell)을 다시 시작하세요.

3. 설치 확인:
   ```bash
   node --version
   npm --version
   ```

### 2. 프로젝트 설정

Node.js 설치가 완료되면 다음 명령어를 실행하세요:

```bash
# ui 폴더로 이동
cd ui

# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

### 3. 브라우저에서 접속

개발 서버가 실행되면 자동으로 브라우저가 열리거나, 수동으로 다음 주소로 접속하세요:
- http://localhost:3000

## 문제 해결

### "npm이 인식되지 않습니다" 오류
- Node.js 설치 후 터미널을 다시 시작했는지 확인하세요.
- 시스템 환경 변수에 Node.js 경로가 추가되었는지 확인하세요.
- Node.js 설치가 완료되었는지 확인하세요.

### 포트 3000이 이미 사용 중입니다
- 다른 애플리케이션이 포트 3000을 사용하고 있을 수 있습니다.
- `vite.config.js`에서 포트 번호를 변경할 수 있습니다.
