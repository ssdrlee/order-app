# Render.com에서 PostgreSQL 데이터베이스 찾기

## 방법 1: 사이드바에서 직접 찾기

1. **Render.com 대시보드 접속**
   - https://dashboard.render.com

2. **왼쪽 사이드바 확인**
   - 사이드바에서 **"Databases"** 또는 **"PostgreSQL"** 섹션 찾기
   - 또는 **"Services"** 섹션에서 데이터베이스 타입 필터링

3. **데이터베이스 목록 확인**
   - 데이터베이스 목록이 표시됩니다
   - 각 데이터베이스의 이름, 상태, 지역 등이 표시됩니다

## 방법 2: "+ New" 버튼을 통해 확인

1. **"+ New" 버튼 클릭**
   - 대시보드 상단 오른쪽에 있는 **"+ New"** 버튼 클릭
   - 또는 **"+ Add new"** 버튼 클릭

2. **"PostgreSQL" 선택**
   - 드롭다운 메뉴에서 **"PostgreSQL"** 선택
   - 또는 **"Database"** → **"PostgreSQL"** 선택

3. **기존 데이터베이스 확인**
   - 새로 생성하지 않고 취소하면
   - 기존 데이터베이스 목록을 볼 수 있습니다

## 방법 3: 검색 기능 사용

1. **검색 바 사용**
   - 대시보드 상단의 검색 바 클릭 (^K 또는 검색 아이콘)
   - "postgres" 또는 "database" 검색
   - 데이터베이스 목록이 표시됩니다

## 방법 4: 서비스 목록에서 필터링

1. **"Ungrouped Services" 섹션 확인**
   - Overview 페이지에서 "Ungrouped Services" 섹션 확인
   - 모든 서비스 목록이 표시됩니다

2. **데이터베이스 타입 필터링**
   - 서비스 목록에서 데이터베이스 타입으로 필터링
   - PostgreSQL 데이터베이스가 표시됩니다

## 데이터베이스가 보이지 않는 경우

### 새로 생성해야 하는 경우

1. **"+ New" 버튼 클릭**
   - 대시보드 상단 오른쪽의 **"+ New"** 버튼 클릭

2. **"PostgreSQL" 선택**
   - 드롭다운 메뉴에서 **"PostgreSQL"** 선택

3. **데이터베이스 생성**
   - 데이터베이스 이름 입력
   - 지역 선택 (예: Oregon)
   - 플랜 선택 (Free 또는 Paid)
   - "Create Database" 클릭

4. **생성 완료 대기**
   - 데이터베이스 생성 완료까지 약 1-2분 소요
   - 생성 완료 후 데이터베이스 페이지로 이동

## 데이터베이스 연결 정보 확인

데이터베이스를 선택한 후:

1. **데이터베이스 페이지 접속**
   - 데이터베이스 이름 클릭

2. **"Connections" 또는 "Info" 탭 확인**
   - 왼쪽 사이드바에서 **"Connections"** 클릭
   - 또는 **"Info"** 탭 클릭

3. **연결 정보 확인**
   - **Internal Database URL**: Render 서비스 내부에서 사용
   - **External Database URL**: 로컬에서 연결 시 사용
   - 또는 개별 정보:
     - Host
     - Port (5432)
     - Database
     - User
     - Password

## 주의사항

- **Free 플랜**: 데이터베이스가 90일 후 자동으로 삭제될 수 있습니다
- **비밀번호**: 데이터베이스 비밀번호는 안전하게 보관하세요
- **연결 정보**: External Database URL은 로컬에서 연결 시에만 사용 가능합니다

## 다음 단계

데이터베이스를 찾은 후:

1. **연결 정보 확인**
   - Connections 탭에서 연결 정보 확인

2. **로컬 .env 파일 설정**
   - `server/.env` 파일에 연결 정보 설정

3. **데이터베이스 초기화**
   - `npm run db:init-render` 실행
